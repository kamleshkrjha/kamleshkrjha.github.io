    Instructions:
  1. Collect gold coins for 1 point.
  2. Collision with red coins will end the game.
  3. Shoot red coins to get 2 points, 1 point gets deducted if you shoot gold
  4. Game will be over if you fail to reach the target for the level.
    Navigations:
  1. left: left arrow.
  2. right: right arrow
  3. jump: up arrow
  4. shoot: space
  5. press Enter key to pause/resume
